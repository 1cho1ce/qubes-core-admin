NAME
====
qvm-backup

:Date:   2012-04-10

SYNOPSIS
========
| qvm-backup [options] <backup-dir-path>

OPTIONS
=======
-h, --help
    Show this help message and exit
-x EXCLUDE_LIST, --exclude=EXCLUDE_LIST
    Exclude the specified VM from backup (might be repeated)
